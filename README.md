# Internship-Project-Electricity-Bill-Generator-
This project is developed using C programming language.
The program calculates electricity bills based on units consumed.
User enters the number of electricity units.
Different rates are applied for different unit ranges.
First 100 units are charged at ₹1.50 per unit.
Units from 101–200 are charged at ₹2.50 per unit.
Units above 200 are charged at ₹4.00 per unit.
The program uses if-else statements for bill calculation.
The float data type is used to store bill amounts.
The project demonstrates conditional logic, arithmetic operations, and user input handling in C.
